Project Goals (Objectives)
Data Analysis: To analyze Electric Vehicle charging data and compare battery performance and driving range.

Visualization: To visualize charging station information and compare different EV brands and models.

Platform Integration: To create interactive dashboards using Tableau, publish them online via Tableau Public, and integrate them into a Flask web application.

Usability: To help users understand complex EV data through simple, meaningful visualizations to support better decision-making.

Target Audience
While the document does not explicitly list specific user personas, the platform is designed for end users who need to understand charging patterns, monitor battery efficiency, compare vehicle performance, and explore driving ranges.

Key Features
Analytical Capabilities: EV data analysis, battery performance analysis, and charging pattern analysis.

Interactive Visuals: Interactive dashboards, map-based reports, charts, graphs, and KPI cards.

Functionality: Dynamic filters, Tableau story creation, and publishing on Tableau Public.

Platform: A user-friendly, Flask-based web application interface.

Tech Stack
Database: MySQL or Oracle SQL for storing and managing datasets.

Data Analysis: SQL for performing analysis and retrieving information.

Visualization: Tableau for creating dashboards and reports; Tableau Public for online publishing.

Web Development: Python (Flask) for the web application, HTML for page structure, and CSS for styling and responsiveness.

Libraries/Tools
The project relies on the following tools:

MySQL/Oracle SQL: Data storage and analysis.

Tableau/Tableau Public: Dashboard creation and hosting.

Python Flask: Web framework.

HTML/CSS: Frontend design.
