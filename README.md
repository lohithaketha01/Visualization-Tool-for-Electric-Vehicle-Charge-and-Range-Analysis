# 🚗 Visualization Tool for Electric Vehicle Charge and Range Analysis

## 📖 Introduction

Electric Vehicles (EVs) are becoming one of the most popular and eco-friendly modes of transportation. As the number of EV users continues to grow, a large amount of data is generated from charging stations, battery performance, driving range, charging time, and vehicle efficiency. Analyzing this information manually is difficult and time-consuming.

This project, **"Visualization Tool for Electric Vehicle Charge and Range Analysis,"** is developed to analyze and visualize Electric Vehicle data using **SQL, Tableau, and Flask**. The project helps users understand charging patterns, compare vehicle performance, monitor battery efficiency, and explore driving range through interactive dashboards. These visualizations make complex data simple, meaningful, and easy to understand.

---

# 🎯 Problem Statement

With the increasing adoption of Electric Vehicles, managing and analyzing large amounts of charging and battery data has become challenging. Manual analysis cannot provide quick insights into vehicle performance, charging behavior, and driving range. Therefore, an interactive visualization system is required to convert raw EV data into meaningful information that supports better decision-making.

# 💻 Technologies Used

This project uses different technologies to collect, analyze, visualize, and display Electric Vehicle data effectively.

🔹 **MySQL / Oracle SQL** is used to store and manage the Electric Vehicle dataset in a structured format.

🔹 **SQL** is used to perform data analysis, retrieve required information, and generate meaningful insights.

🔹 **Tableau** is used to create attractive dashboards, charts, maps, and visual reports.

🔹 **Tableau Public** is used to publish the dashboards online for easy access and sharing.

🔹 **Python Flask** is used to develop the web application and integrate Tableau dashboards into the website.

🔹 **HTML** is used to design the structure of the web pages.

🔹 **CSS** is used to improve the appearance of the website by adding colors, layouts, and responsive designs.

Together, these technologies help transform raw Electric Vehicle data into an interactive and user-friendly analytics platform.

# ⚙️ Working of the Project

### 📌 Step 1: Data Collection

The Electric Vehicle dataset containing information about charging stations, battery capacity, charging time, vehicle models, and driving range is collected from reliable sources.

### 📌 Step 2: Data Storage

The dataset is imported into MySQL or Oracle SQL Database for secure storage and easy management.

### 📌 Step 3: Data Analysis

SQL queries are used to analyze the dataset by calculating totals, averages, filtering records, grouping data, and generating useful insights.

### 📌 Step 4: Dashboard Creation

The SQL database is connected to Tableau to create interactive dashboards such as:

* Charging Station Analysis
* Battery Performance
* Driving Range Comparison
* EV Brand Comparison
* Charging Pattern Analysis
* Maps and KPI Cards

### 📌 Step 5: Dashboard Publishing

The dashboards and stories are published using Tableau Public.

### 📌 Step 6: Web Application

The published Tableau dashboards are embedded into a Flask web application where users can interact with the dashboards through a simple website.
